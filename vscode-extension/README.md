# NewPNG VS Code extension

Language support for `.npng` design source files.

## Features

- Registers `.npng` files as the `npng` language.
- Reuses YAML highlighting.
- Adds a starter npng snippet.
- Adds `NewPNG: Open Online Viewer`, which opens the current file in the hosted viewer with the source embedded in the URL hash.

## Development

```bash
npm install
npm run compile
```
